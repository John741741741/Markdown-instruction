# Инструкция по работе с Markdown

## Выделение текста

Чтыбы выделить текст курсивом, необходимо обромить его звёздочкой (*). Например, *вот так*

Чтобы выделить текст полужирнфм, необходимо добавить двойные звёздочки (**). Например, **вот так**
